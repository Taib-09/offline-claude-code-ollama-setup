# Offline Claude Code + Ollama Setup
This project documents how to build a local, privacy‑friendly AI development environment inspired by Claude Code, using:
- Ollama for running local models
- VS Code or terminal workflows
- Scripts and guides to avoid region, pricing, and reliability issues
## Goals
- Help developers who cannot reliably access commercial AI tools
- Provide step‑by‑step setup instructions for local AI coding workflows
- Share troubleshooting notes for common installation and PATH issues
- Offer lightweight solutions that work on modest hardware
## What this repo includes
- Installation steps for Ollama and required tools
- Example workflows similar to Claude Code (code generation, refactoring, debugging)
- Tips for dealing with CLI, PowerShell, Node.js, and environment variables
- Best practices for keeping the system fast and stable
## Why this matters
Many developers face constraints:
- Limited budgets for paid AI subscriptions
- Unreliable access to cloud AI tools
- Older or lower‑spec laptops
This repo aims to make AI‑assisted coding more accessible by focusing on local, open, and affordable setups.
